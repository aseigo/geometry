A set of geometry types for WKT/WKB, EWKT/EWKB and GeoJson.

## Installation

The package can be installed by adding `geometry` to your list of
dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:geometry, "~> 0.3"}
  ]
end
```
